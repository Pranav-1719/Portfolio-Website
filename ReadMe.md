# Portfolio Website

A personal portfolio website designed to showcase my skills, projects, education, certifications, and professional journey. The website serves as a central hub for recruiters, collaborators, and fellow developers to learn more about my work and achievements.

## Live Demo

Visit the website here:

**https://pranav-1719.github.io/Portfolio-Website/**

## Features

* Responsive and mobile-friendly design
* Professional portfolio layout
* About Me section
* Skills and technologies showcase
* Project portfolio with descriptions
* Resume download option
* Contact information and social links
* Clean and modern UI

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/Pranav-1719/Portfolio-Website.git
```

### Run Locally

1. Navigate to the project directory.
2. Open `index.html` in your browser.

## Customization

You can easily customize the portfolio by:

* Updating personal information in `index.html`
* Modifying styles in `style.css`
* Replacing assets in the `images` folder
* Updating `Resume.docx` with the latest version of your resume
